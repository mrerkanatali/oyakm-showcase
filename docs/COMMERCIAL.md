# Commercial models (draft)

**Legal review required before use in contracts.**

---

## English

### Model A — License + initial setup + paid enhancements

- **License** is granted **with agreed modules** (feature bundle) as a complete package.
- **Initial setup** is included (deployment to buyer-approved hosting, baseline configuration).
- **Further development**, new features, or out-of-scope changes are **quoted separately**.

### Model B — Setup fee + monthly support (no source delivery)

- **Initial setup fee** applies.
- **Application source code is not delivered** to the buyer; operation is consistent with hosted or vendor-operated deployment (exact terms to be defined in contract).
- **Monthly software support fee** (similar in rhythm to recurring infrastructure billing).
- **Excluded** from that fee (buyer or separate line items): domain registration, hosting charges, third-party store fees, Apple/Google developer accounts, etc. — unless explicitly included in a written addendum.

### General

- Both models can be positioned as alternatives; final SLA, IP, and liability clauses require counsel.
- Public showcase materials remain documentation-only and do not represent source code delivery or implementation transfer.

## Türkçe

### Model A — Lisans + ilk kurulum + geliştirme

- **Lisans**, üzerinde anlaşılan **modüllerle birlikte** paket olarak verilir.
- **İlk kurulum** dahildir (alıcı onaylı hosting’e kurulum, temel yapılandırma).
- **Sonraki geliştirmeler**, yeni özellikler veya kapsam dışı işler **ayrı ücretlendirilir**.

### Model B — Kurulum + aylık destek (kod teslimi yok)

- **İlk kurulum ücreti** alınır.
- **Uygulama kaynak kodu alıcıya verilmez**; barındırma veya işletmeci modeli sözleşmede net tanımlanır.
- **Aylık yazılım destek ücreti** (domain/hosting faturalamasına benzer periyot).
- Bu ücrete **dahil olmayan** kalemler: domain, hosting, mağaza ücretleri, Apple/Google geliştirici hesapları vb. — aksi yazılı ek protokolle belirtilmedikçe alıcı veya ayrı kalem.

### Genel

- İki model alternatif sunulabilir; SLA, fikri mülkiyet ve sorumluluk maddeleri için **hukuk danışmanlığı** şarttır.
- Public showcase içeriği yalnız tanıtım/dokümantasyon amaçlıdır; kaynak kod teslimi veya implementasyon devri anlamına gelmez.
