# Calculation tools (high-level description only)

**No source code** is provided in this showcase repository.

---

## English

The mobile application includes **financial and utility calculators** aimed at members, for example:

- **Salary / compensation-related** estimations (categories aligned with public-sector style use cases).
- **Inflation or purchasing-power style** illustrative tools.
- **Deposit / loan style** calculators with clear non-advisory disclaimers in the product UI.

Each module is implemented in the **mobile client** with validation and UX appropriate to Turkish locale; **formulas and implementation details are proprietary** and are not part of this documentation pack.

---

## Türkçe

Mobil uygulamada **finans ve yardımcı hesaplama** modülleri bulunur; örnek başlıklar:

- **Maaş / ödeme** ile ilgili tahmin araçları (kamu personeli senaryolarına uygun kategoriler).
- **Enflasyon / satın alma gücü** örneklemeleri.
- **Mevduat / kredi** tipi hesaplayıcılar; ürün arayüzünde **yatırım tavsiyesi değildir** uyarıları.

Her modül **mobil istemcide** Türkçe yerelleştirme ile çalışır; **formül ve uygulama detayları ticari sır** kapsamında olup bu pakette yer almaz.
